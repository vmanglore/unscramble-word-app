# Purpose

The SEO Opportunity Matrix helps prioritize future page types and features based on:

* Search demand
* Ranking difficulty
* User value
* Strategic value
* Long-term platform alignment

Not all traffic opportunities are equal.

The goal is not simply to attract visitors.

The goal is to attract visitors who benefit from the platform and are likely to return.

---

# Scoring Framework

Each opportunity is evaluated across five dimensions.

---

## Search Demand

Estimated volume of relevant searches.

Scale:

```text
1 = Very Low
5 = Very High
```

---

## Ranking Difficulty

How difficult it is to rank.

Scale:

```text
1 = Easy
5 = Difficult
```

Lower scores are better.

---

## User Value

How useful the page is to users.

Scale:

```text
1 = Low
5 = High
```

---

## Strategic Value

Alignment with long-term platform goals.

Scale:

```text
1 = Weak
5 = Strong
```

---

## Priority Score

Overall implementation priority.

Scale:

```text
1 = Low
5 = High
```

# Strategic Observations

Traffic alone should not drive roadmap decisions.

The most valuable opportunities are those that satisfy all of the following:

* Strong search demand
* High user value
* Strong strategic alignment
* Reusable data structures

Examples:

```text
Word Finder
Word Pages
Knowledge Layer
```

These opportunities contribute to both traffic growth and long-term platform differentiation.

---

# Success Criteria

This matrix succeeds when it helps answer:

1. What should we build next?
2. Which pages create the most value?
3. Which pages support the long-term vision?
4. Which opportunities are distractions?
5. Which opportunities strengthen the competitive moat?

The platform should prioritize opportunities that build durable assets rather than temporary traffic spikes.

---

# Guiding Principle

Do not chase every keyword.

Build page types that create lasting value, reusable knowledge, and a stronger platform over time.

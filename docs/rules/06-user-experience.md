# User Experience Rules

UX-001
Searches should feel instantaneous.

Target:
< 1 second

UX-002
Mobile support is mandatory.

UX-003
Clear feedback should be provided when no results exist.

UX-004
Users should be able to start a new search easily.

UX-005
Interfaces should remain simple and intuitive.

UX-006
Advanced options should not overwhelm casual users.

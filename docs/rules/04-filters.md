# Filter Rules

FLT-001
Length filter is optional.

FLT-002
Length filter returns only words of that length.

FLT-003
Starts With filter is optional.

FLT-004
Ends With filter is optional.

FLT-005
Contains filter is optional.

FLT-006
Blank filters must be ignored.

FLT-007
Multiple filters are combined using AND logic.

FLT-008
Filter behavior must remain stable across releases.

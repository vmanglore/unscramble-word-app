# Data and Dictionary Rules

DD-001
Dictionary quality directly impacts result quality.

DD-002
The approved dictionary source is the authority for whether a word is valid.

DD-003
Dictionary updates must not reintroduce invalid/generated/non-dictionary strings.

DD-004
Valid words from the approved dictionary source should remain available even when they are uncommon.

DD-005
Abbreviations, acronyms, and fragments must be handled according to the approved dictionary/source policy, not guessed by implementation.

DD-006
Data changes should be validated through regression tests.

DD-007
Search indexes must remain consistent with source data.
